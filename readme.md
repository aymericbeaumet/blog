# aymericbeaumet.com

Personal site of Aymeric Beaumet — posts, talks, and projects. Built with
[Hugo](https://gohugo.io/) and deployed to GitHub Pages by the workflow in
`.github/workflows/`. The `main` branch is the source of truth; every push
triggers a build + deploy.

## Local development

```sh
npm install
npm run dev      # hugo server on :1313 with drafts
npm run build    # production build into ./public
npm run serve    # serve ./public with no fast-render
```

Hugo (extended) is pulled in via npm — no separate install needed. The
`prebuild`/`predev`/`preserve` hooks copy vendored assets
(`@fontsource/lato`, `instant.page`) into `static/fonts/` and `assets/js/`
respectively.

## Content layout

```
content/
  posts/<slug>/index.md       articles
  talks/<slug>/index.md       talks + slides
  projects/<slug>/index.md    project descriptions
  drafts/<slug>/index.md      unpublished — excluded from production
```

Each entry is a page bundle: the `index.md` plus any page resources
(hero image, attachments named `*--*` show up under "Attachments").

## Notable features

- Responsive `<picture>` (AVIF + WebP, 1x/2x) generated at build time by
  Hugo's image pipeline (`layouts/partials/production-picture.html`).
- Dark mode via `prefers-color-scheme`.
- RSS feed at `/feed.xml`.
- `https://aymericbeaumet.com/<slug>.{md,markdown,txt,raw,source}` 302's
  to the canonical GitHub source of that page (generated by
  `scripts/copy-markdown.js`).
- Comments via [Utterances](https://utteranc.es/) on every page; a few
  legacy posts also embed Disqus.
